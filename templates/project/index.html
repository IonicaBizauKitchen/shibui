Typography

Shibui can pretty much style anything Markdown can render and makes sure your page has a consistent line-height. You can learn about the markup by inspecting the elements on this page.

Inline Elements

You can link, emphasize with the strong tag and the em tag, or wrap inline snippets of code with the code tag.

Feel free to use <b> and <i> in HTML5, but their usage has changed a bit. <b> is meant to highlight words or phrases without conveying additional importance while <i> is mostly for voice, technical terms, etc.

Even though Markdown doesn't support abbr, Shibui does. Abbreviations with a title attribute have a light dotted bottom border and a help cursor on hover.
